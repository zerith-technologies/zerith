# Documentação do Projeto ZERITH

Bem-vindo à pasta de documentação do monorepo ZERITH! Aqui você encontra todos os registros, decisões, diagramas, processos e materiais de apoio do projeto.

## Índice

- [ADRs](adr/) — Decisões de arquitetura (Architecture Decision Records)
- [Diagramas](diagrams/) — ER, UML, fluxos, arquitetura, etc.
- [Onboarding](onboarding/) — Guias para novos membros e integração
- [Sprints](sprints/) — Planejamento, retrospectivas e lições aprendidas
- [Especificações](specs/) — Documentos técnicos e funcionais
- [Processos](processos/) — Políticas, convenções, rotinas e padrões internos

## Recomendações de uso
- Sempre que criar um novo documento, adicione um link aqui no índice.
- Mantenha os arquivos organizados nas subpastas temáticas.
- Prefira Markdown (.md) para facilitar leitura e versionamento.
- Use nomes de arquivos e títulos claros e objetivos.

## Sobre a documentação
Esta estrutura visa facilitar a colaboração, o onboarding e a rastreabilidade das decisões e processos do projeto ZERITH.
